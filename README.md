# 🔍 Finansal Anomali Tespit Sistemi

AI destekli finansal dolandırıcılık ve anomali tespit uygulaması.

## 📌 Ne Yapıyor?
- Finansal işlem verilerini analiz eder
- Şüpheli ve anormal işlemleri otomatik tespit eder
- Sonuçları interaktif grafiklerle görselleştirir
- Gece saatlerinde yapılan yüksek tutarlı işlemleri işaretler

## 🛠️ Kullanılan Teknolojiler
- **Python**
- **Scikit-learn** — Isolation Forest algoritması
- **Streamlit** — Web arayüzü
- **Pandas** — Veri işleme
- **Plotly** — İnteraktif grafikler

## 🚀 Nasıl Çalıştırılır?
pip install pandas scikit-learn plotly streamlit
python -m streamlit run anomali.py

## 💡 Kullanım Alanları
- Banka dolandırıcılık tespiti
- Kurumsal finansal risk analizi
- Şüpheli işlem raporlama
